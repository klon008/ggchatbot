# CloudPub CLI (bundled)

Скопированный клиент CloudPub для туннеля Album API.

- Бинарь: `clo.exe` (обновлять: `.\clo.exe upgrade` из этой папки)
- Бот по умолчанию использует этот путь (`CLO_EXE_PATH` необязателен)
- Перед `publish` бот пишет токен: `clo set token <CLO_TOKEN из .env>`

Ручной тест: `installer\tunnel-testing.cmd`
