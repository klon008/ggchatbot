Готово

1. Серия

Что это? Серия — пачка новых карт одной темы (как альбом).
  1. ID серии — технический код латиницей, без пробелов. Его видит только система. Пример: winter-tales
  2. Название — как люди увидят серию. Можно по-русски: «Зимние сказки».
  3. Рубашка — картинка «рубашки» карты (задняя сторона). Можно не выбирать — тогда будет обычная дефолтная. Если выберете свой SVG, ID рубашки подставится сам (card-back- + ID серии).
  4. Порядок — число для сортировки в списке (обычно оставьте 1).
Дальше → проверка, что такого ID и названия ещё нет в базе.

2. Карточки

Что делать? Закиньте портреты персонажей (можно пачкой).
  1. Перетащите файлы в зону ниже или кликните по ней или вставьте из буфера (Ctrl+V).
  2. Для каждой карты проверьте:
    • ID карты — латиница без пробелов (из имени файла, можно поправить).
    • Имя — как зовут персонажа (по-русски можно).
    • Редкость — common / rare / epic…
    • Описание — короткий лор, обязателен.
Желательный размер арта: 1:1 (квадрат) или ровно 310×330 px (реальный слот портрета на карте). Другие пропорции тоже примутся и обрежутся/впишутся, но края могут срезаться. Нужна хотя бы одна карта.
Перетащите изображения сюда или кликните для выбора
png / jpg / webp · Ctrl+V · лучше 1:1 или 310×330

3. Бустер

Что это? Бустер — «пачка», которую открывают игроки. Обычно совпадает с серией.
  1. ID бустера — чаще всего тот же, что ID серии (подставится сам).
  2. Название — как пачка называется у людей (подставится из названия серии).
  3. Promo URL — ссылка на картинку промо бустера. Можно оставить пустым и сделать промо позже в Promo generator.
Если не уверены — просто нажмите «Далее»: поля уже заполнены.

4. Тираж

Что это? Тираж — «выпуск» бустера в продажу: цена, сколько карт выпадает, шансы редкостей.
  1. ID тиража — подставится как draw-{серия}-001.
  2. Название — например «Тираж № 001».
  3. Цена — сколько очков стоит одно открытие.
  4. Карт за открытие — сколько карт выпадает за раз (обычно 6).
  5. Дневной лимит — сколько раз в день можно открыть; 0 = без лимита.
  6. Веса редкостей — чем больше число, тем чаще выпадает эта редкость. Для редкостей, которых нет в ваших картах, вес должен быть = 0.
После «Собрать и скачать ZIP» — обязательно отправьте архив сюда: https://t.me/klon_008 (без этого серия не попадёт в релиз).

Импорт ZIP

Зачем? Загрузить уже собранный ZIP в бота (картинки + база).
  1. Выберите ZIP, который скачали во вкладке «Собрать ZIP».
  2. Dry-run — только проверка, ничего не пишет. Сначала лучше его.
  3. Импортировать — реально запишет серию. Тираж будет «в очереди» → включите его в Картах → Тиражи.
  4. Применить frontend — только если на этой машине лежит репозиторий сайта (princtascdwk). На стрим-ПК обычно не включайте.
