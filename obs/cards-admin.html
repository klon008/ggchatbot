Готово

Основные - глобальные переключатели модуля карт и справочник чат-команд. Здесь включают/выключают ответы на !бустер / !альбом и задают дневной лимит открытий (если у тиража свой лимит > 0 - он важнее). OBS-анимация открытия: Browser Source /booster.html (~384×560, прозрачный фон).

Глобальные настройки

При выключенном модуле команды ниже отвечают отказом. Скорость анимации: 1.0 = норма, 2.0 = в 2 раза быстрее, 0.5 = в 2 раза медленнее.

Команды чата

команда описание
!бустер Открыть тираж: анонс в чат + анимация OBS (/booster.html) + итог
!бустер инфо Активный тираж: карты, пул, цена и promo-ссылка
!альбом Прогресс коллекции и ссылка на альбом на сайте

Коллекции - полный каталог карт из БД: превью, редкость, серия, лор. Только просмотр. Арты лежат в obs/assets/cards/{slug}.webp; описания читаются из cardDetails.json (сайт / update.cmd), из админки не правятся.

Бустеры - набор (пул) карт, из которого тираж выдаёт дроп. Создайте бустер, отметьте карты чипами; «Показать состав» - галерея, «Переименовать» сохраняет имя. Promo - ссылка на картинку для стрима (https://… или /assets/…); «Генерировать» открывает 3D-сцену (/promo-generator.html?booster=…) — скачайте JPG, загрузите на imgbb и вставьте URL сюда. Бустер ≠ серия: в одном пуле могут быть карты из разных серий.

Состав

Тиражи - конкретный запуск бустера для зрителей: цена, число карт, шансы редкости. Очередь FIFO (первый созданный - первый на выход), одновременно один live.

  • queued - в очереди; «Активировать» только у первого, если нет live
  • active - текущий: «Пауза» или «Завершить»
  • paused - пауза: «Возобновить» или «Завершить»
  • closed - завершён навсегда, активировать нельзя (нужна «Копия» в конец очереди)
«Завершить» поднимает следующий из очереди. «Сразу активировать» при создании закрывает текущий live. Шансы задаются только при создании; у существующих - кнопка «Шансы» (просмотр).

Создать тираж

Шансы редкости: относительные веса (сумма не обязана быть 100). Проценты - для наглядности.
