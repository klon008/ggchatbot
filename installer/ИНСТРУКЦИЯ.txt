================================================================================
  GGCHATBOT - ИНСТРУКЦИЯ ДЛЯ СТРИМЕРА
================================================================================

В папке установщика три главных файла (запуск двойным щелчком):

  install.cmd  - установка (один раз в начале)
  start.cmd    - запуск бота перед стримом
  update.cmd   - обновление кода бота (когда разработчик сказал, что есть новая версия)


--------------------------------------------------------------------------------
  ПЕРВАЯ УСТАНОВКА (один раз)
--------------------------------------------------------------------------------

1. Скачайте архив установщика у разработчика (GitHub Releases)
   и распакуйте, например в:  C:\ggchatbot

2. Дважды нажмите:  install.cmd  (в распакованной папке)

   Скрипт скачает код бота в подпапку ggchatbot\ — туда заходить не нужно.

3. Если спросит "Установить Git?" - нажмите  y  и Enter

4. Если попросит установить Python:
   - откройте https://www.python.org/downloads/
   - скачайте и установите Python
   - ОБЯЗАТЕЛЬНО поставьте галочку "Add python.exe to PATH"
   - после установки снова запустите install.cmd

5. Когда установка закончится - напишите разработчику.
   Он настроит файл .env (логин/пароль бота GoodGame).

6. (Опционально) Настройка баланса под свой канал:
   - ggchatbot\bot\princess\settings.py  — принцессы, кража, дейлики, кубик
   - ggchatbot\bot\song_request\settings.py  — стоимость !sr (SR_COST)
   - ggchatbot\bot\roulette\settings.py  — рулетка
   - ggchatbot\bot\minigames\settings.py  — общая казна мини-игр
   - ggchatbot\bot\races\settings.py  — забеги (!забег)
   При install.cmd эти файлы создаются из *.example.py.
   Они НЕ перезаписываются при update.cmd.
   После обновления сравните *.example.py с вашим settings.py,
   если в репозитории появились новые параметры.

7. Настройте OBS (один раз):
   - источник "Браузер"
   - URL:  http://127.0.0.1:8765/player.html
   (подробности - у разработчика)


--------------------------------------------------------------------------------
  ПЕРЕД КАЖДЫМ СТРИМОМ
--------------------------------------------------------------------------------

1. Запустите OBS
2. Дважды нажмите:  start.cmd
3. Должно открыться чёрное окно с текстом "Запуск бота..."
   НЕ ЗАКРЫВАЙТЕ это окно во время стрима!

Чтобы остановить бота - закройте это окно крестиком.


--------------------------------------------------------------------------------
  ОБНОВЛЕНИЕ (когда разработчик написал "есть обновление")
--------------------------------------------------------------------------------

1. Закройте окно бота (start.cmd), если оно открыто
2. Дважды нажмите:  update.cmd
3. Дождитесь "Обновление завершено" (база bot.db обновится автоматически)
4. Снова запустите:  start.cmd

При обновлении кода база данных обновляется сама — вручную SQL выполнять
не нужно. Если update.cmd пишет про ошибку миграции — закройте бота и
повторите update.cmd.

Если разработчик прислал новый архив установщика - распакуйте поверх
старых файлов (install.cmd, start.cmd, …) или в новую папку и перенесите
папку ggchatbot\ с настройками.


--------------------------------------------------------------------------------
  МИГРАЦИЯ СО СТАРОГО БОТА (JSON -> SQLite, один раз)
--------------------------------------------------------------------------------

Если переносите данные со старой версии бота:

1. Положите JSON-файлы в папку ggchatbot\data\  (разработчик подскажет какие)
2. Дважды нажмите:  migrate.cmd
3. Подтвердите  y  в окне
4. После успеха запустите start.cmd как обычно


--------------------------------------------------------------------------------
  ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ
--------------------------------------------------------------------------------

- Бот молчит в чате        -> напишите разработчику (проверит .env)
- Окно сразу закрывается   -> запустите install.cmd ещё раз
- Ошибка при update.cmd    -> напишите разработчику, пришлите скрин окна
- Не играет музыка в OBS   -> проверьте URL браузера в OBS (см. выше)

НЕ редактируйте остальные файлы бота вручную - только install / start / update.
Исключение: settings.py для баланса (см. раздел установки выше).
